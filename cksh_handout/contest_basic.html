遞迴

在函式中再次呼叫函式稱為遞迴

在實做遞迴時 需要格外小心中止條件

以免發生stack overflow停不下來

遞迴範例

回傳k!

int muti(int a){
    if(a==1) return 1;
    else return a*muti(a-1);
}
int main{
    int a;cin>>a;
    int answer=muti(a);
    cout<<answer<<endl;
}

下頁有講解

講解(?)

在muti函式中我不斷讓a*muti(a-1)

所以會不斷的進到新的muti函式->muti(a-1)

直到a=1 因為muti(1)計算完了

因此muti(2)拿到muti(1)的答案而計算2*1結束

最終往上完成muti(a)的遞迴計算，回傳a!到main
